# Algorithms and Data Structures - Merge Sort


## Characteristics
- Time complexity:
    - Best: Ω(n log(n))
    - Average: Θ(n log(n))
    - Worst: O(n log(n))
- Space complexity: O(n)
- Not in-place
- Stable
